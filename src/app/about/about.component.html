About this TODO app

The Todo App built with Angular is a simple yet effective tool for managing tasks. It allows users to add, delete, and mark tasks as completed. The app provides a clean and intuitive user interface, making it easy to organize tasks efficiently.

Technologies Used:
  • Angular Framework: The app is built using Angular, a powerful JavaScript framework for building web applications.
  • HTML/CSS: Standard web technologies are used for structuring the user interface and styling elements.
  • Bootstrap: Bootstrap is used to enhance the visual appeal and responsiveness of the app, providing pre-designed components and layouts.
Features:
  • Adding Tasks: Users can add new tasks by providing a title and description. This feature ensures that users can capture all necessary details about a task.
  • Completing Tasks: Tasks include a checkbox that users can mark to indicate completion. This functionality allows users to track their progress and distinguish between completed and pending tasks.
  • Strikethrough Effect: Completed tasks are visually differentiated by a strikethrough effect applied to the task title and description. This visual cue enhances usability by clearly indicating which tasks have been completed.
  • Deleting Tasks: Each task includes a delete button, allowing users to remove tasks that are no longer needed. This feature promotes task management flexibility.
